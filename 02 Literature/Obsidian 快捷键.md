---

- 插入模板
```c
control + shift + t
```
- 中英文格式化
```c
control + shift + s
```
- 创建新的文件
```c
command + n
```
- 创建脚标
```c
control + shift + 6
```
- 打开/关闭 左/右侧边栏
```c
command + shift + <-
command + shift + ->
```
- 打开设置
```c
ctrl + ,
```
